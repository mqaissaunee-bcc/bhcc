NETW 112: Containers & Orchestration

Containerization with Podman & OpenShift Administration

Instructor: TBD Prereq: NETW 110 100% Hands-on Lab

Course Description

This course introduces students to modern application deployment using container technology and orchestration platforms. Designed for IT students with basic Linux CLI experience, it provides a hands-on path from container fundamentals to enterprise-scale orchestration using **Podman** and **Red Hat OpenShift**.

Red Hat Academy

Prepares you for EX188 (Containers) and provides foundational skills for EX280 (OpenShift Admin).

Assessment

No written exams. Mastery is demonstrated through live lab practicals in the Red Hat environment.

Learning Outcomes

  • ✔ Run, manage, and troubleshoot containers using Podman on RHEL.
  • ✔ Build custom images using Containerfiles and multi-stage builds.
  • ✔ Persist data using volumes and bind mounts with correct SELinux configuration.
  • ✔ Navigate OpenShift clusters using the web console and oc CLI.
  • ✔ Manage applications using Kubernetes Pods, Deployments, and Services.
  • ✔ Configure reliability with liveness/readiness probes and Horizontal Pod Autoscaling.

Required Materials

Red Hat Learning Subscription

Access to the Red Hat Online Learning Environment (OLE) is provided through the Red Hat Academy. This includes:

  • DO188: Introduction to Containers with Podman
  • DO180: OpenShift Administration I
Tech Stack

RHEL 9, Podman 4.x, Buildah, Skopeo

Environment
